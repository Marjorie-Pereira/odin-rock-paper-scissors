# The Odin Project - Rock, Paper, Scissors
In this project, I practiced what is teached in the course curriculum about JavaScript Basics, up to the point of making this project.
The main skills demonstrated here are:
- Linking the script in the `index.html` file, using the `<script>` tag.
- Creating functions and passing arguments to them.
- The use of `Math` object methods to get a random number.
- The `switch... case` to process the computer choice.
- Getting user input with `prompt()`.
- The use of logical and comparison operators.
- Ternary operator to decide the winner.
- String interpolation with the use of backticks.

### Bonus
I also made the use of a `for` loop to play 5 rounds, since I have some experience with JavaScript.
